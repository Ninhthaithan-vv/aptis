# Listening Part 1 - Meo nho tung cau

Dung file nay de hoc nhanh:
- Moi cau co 1 moc nho rieng.
- Uu tien nho "tu khoa hinh anh" truoc, roi moi nho dap an.
- Neu gap cau giong nhau, hay nho theo boi canh khac nhau o phan meo.

## Cach hoc nhanh 3 vong

- Vong 1: che dap an, chi doc cau hoi va tu khoa meo.
- Vong 2: doc cau hoi, tu noi dap an trong 2 giay.
- Vong 3: tron cac cau de nham lan nhat nhu thoi gian, dia diem, gia tien.
